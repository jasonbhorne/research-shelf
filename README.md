# The Research Shelf

A growing library of deep dives, companion guides, and curiosity-driven research, all produced using [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview)'s multi-agent research pipeline.

This repository contains 27 research reports spanning K-12 education policy, AI in education, district operations, history, and whatever else caught my attention. Each report was generated by spawning 4-5 parallel research agents that independently gather sources from different categories (government, academic, news, industry), then cross-referencing their findings, verifying URLs, and synthesizing everything into a publication-ready document.

## How It Works

The pipeline is built on two Claude Code skills that orchestrate multi-agent research:

### `/deep-research` (4 agents)
Spawns four agents, each assigned a distinct source category (government, news, academic, industry). Agents research independently, then findings are cross-referenced in a claim matrix, URLs are verified, and confidence scores are aggregated. Produces a .docx with a source reliability matrix.

### `/research-report` (3-5 agents)
A unified pipeline that adds a case-studies agent, a fact-check pass, and optional slide/infographic generation. Supports `--education` mode for Tennessee-specific policy context and `--light` mode for faster, 3-agent runs.

Both skills are in the [`skills/`](skills/) directory and can be dropped into any Claude Code setup.

### Generation Scripts

The [`scripts/`](scripts/) directory contains the Python scripts (using `python-docx`) that generate the final formatted .docx files. These are bespoke per report, showing the full end-to-end pipeline from research findings to publication-ready document.

## Using the Skills

To use these research skills in your own Claude Code setup:

1. Copy the skill files from `skills/` into your `.claude/skills/` directory
2. Update the file paths in the skills to match your environment
3. Run `/deep-research <your topic>` or `/research-report <your topic>`

The skills require Claude Code with agent capabilities (Team, Task, and Agent tools). They work best with models that support parallel tool use.
